Este é um exercício de lógica de programação com conceitos básicos de orientação à objetos

É um protótipo que simula um Cadastro escolar de alunos e professores, onde são coletados os dados pessoais, modelados em Hashes e
inseridos numa lista com buscas precisas utilizando validação de entrada de dados do usuário com dados existentes nas listas.
É possível também editar dados existentes nas listas.

***Isto é um protótipo de iniciante na programação***
